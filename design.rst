Flow design
===========

Receive an email from legacy email protocl (MX to DEFMS)
--------------------------------------------------------

.. code::

        .------.
        | IMAP |
        '------'
           |
          \ /
           '
    .---------------. 
    | defms-gateway | 
    '---------------' 
           |
          \ /
           '
    .---------------. 
    | defms mailbox | 
    '---------------' 
           |
          \ /
           '
    .--------------. 
    | defms-daemon | 
    '--------------' 
           |
          \ /
           '
        .------. 
        | IPFS | 
        '------' 


Send an email to legacy email system (DEFMS to MX)
--------------------------------------------------

.. code::

    .--------------.      
    | defms-daemon |  ---------------. 
    '--------------'                 |
           |                         | 
          \ /                       \ /
           '                         ' 
    .---------------.         .---------------.
    | defms-gateway |         | defms-sentbox |
    '---------------'         '---------------'
           |                         |    
          \ /                       \ /   
           '                         '    
        .------.                  .------.
        | SMTP |                  | IPFS |
        '------'                  '------'

           '
    .---------------.

           |
          \ /
           '
    .--------------.

    '--------------'
